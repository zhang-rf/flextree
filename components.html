Get Started

components provide easy-to-use feature enhancement, we have already developed some frequently used components. And it's very easy to develop your own components, just have a look at these pre-developed components' source code and you will know how to develop yours.

now just have a quick tutorial to learn how to use components.

  1. Components rely on flextree.js so you must download FlextreeJS and setup flextree.js first.
  2. Include flextree.css
    <link rel="stylesheet" href="flextree.css">
  3. Iinclude flextree.components.js
    <script src="flextree.components.js"></script>
  4. Then the only difference is to adding "args.components" option, here use a component "selectable" as a example.
    <div id="demo"></div>
<script src="jquery.js"></script>
<script src="flextree.js"></script>
<script src="flextree.components.js"></script>
<script>
    $(function () {
        var template = '<a href="javascript:;">{myAwesomeText}</a>';
        var dataSource = [{id: 'A1', parent_id: null, myAwesomeText: 'I am A1', myAwesomeData: 'AwesomeA1'},
            {id: 'A2', parent_id: 'A1', myAwesomeText: 'I am A2', myAwesomeData: 'AwesomeA2'},
            {id: 'A3', parent_id: 'A1', myAwesomeText: 'I am A3', myAwesomeData: 'AwesomeA3'},
            {id: 'B1', parent_id: null, myAwesomeText: 'I am B1', myAwesomeData: 'AwesomeB1'},
            {id: 'C1', parent_id: null, myAwesomeText: 'I am C1', myAwesomeData: 'AwesomeC1'},
            {id: 'C2', parent_id: 'C1', myAwesomeText: 'I am C2', myAwesomeData: 'AwesomeC2'}];
        var args = {components: {selectable: [true]}};
        // components format is: {componentName1: [arg1, arg2 ....], componentName2: [arg1, arg2 ....], ...}

        $('#demo').tree(dataSource, template, args);
    });
</script>
  5. Also, there is a global property "$.flextree.components.global".
    $.flextree.components.global = {selectable: [true]}; // for example

Stylesheet

"flextree.css" contains styles that used by components, you can modify styles as you like.

Components

  • Selectable
    this component makes your nodes selectable.

    Arguments

    • multiple {boolean}
      set to true to allow multiple selection, default is false
    • cancelable {boolean}
      whether a node can be deselected, default is true

    Events

    • $.flextree.components.selectable.eventSelect
      this event fires when a node get selected, you can catch this event either on tree or on nodes.

    APIs

    • $().treeSelected()
      get selected nodes of this tree.
    • $().treeSelect()
      select this node.
            var template = '<a href="javascript:;">{myAwesomeText}</a>';
        var dataSource = [{id: 'A1', parent_id: null, myAwesomeText: 'I am A1', myAwesomeData: 'AwesomeA1'},
            {id: 'A2', parent_id: 'A1', myAwesomeText: 'I am A2', myAwesomeData: 'AwesomeA2'},
            {id: 'A3', parent_id: 'A1', myAwesomeText: 'I am A3', myAwesomeData: 'AwesomeA3'},
            {id: 'B1', parent_id: null, myAwesomeText: 'I am B1', myAwesomeData: 'AwesomeB1'},
            {id: 'C1', parent_id: null, myAwesomeText: 'I am C1', myAwesomeData: 'AwesomeC1'},
            {id: 'C2', parent_id: 'C1', myAwesomeText: 'I am C2', myAwesomeData: 'AwesomeC2'}];
        var args = {id: 'id', parentId: 'parent_id', components: {selectable: [true]}};

        $('#demo-selectable').tree(dataSource, template, args, function (data, i) {
            $(this).on($.flextree.components.selectable.eventSelect, function () {
                var selected = 'You selected: ';
                $(this).treeSelected().each(function () {
                    selected += $(this).text() + ' ';
                });
                alert(selected);
            });
        });

  • Checkbox
    this component adds checkbox to your nodes.

    Arguments

    • multiple {boolean}
      set to true to allow multiple check, default is true

    Template
    this component requires some specific elements in your template.

    • <input class="ft-checkbox" type="checkbox">

    Events

    • $.flextree.components.checkbox.eventCheck
      this event fires when a node get checked, you can catch this event either on tree or on nodes.

    APIs

    • $().treeChecked()
      get checked nodes of this tree.
            var template = '<input class="ft-checkbox" type="checkbox"><a href="javascript:;">{myAwesomeText}</a>';
        var dataSource = [{id: 'A1', parent_id: null, myAwesomeText: 'I am A1', myAwesomeData: 'AwesomeA1'},
            {id: 'A2', parent_id: 'A1', myAwesomeText: 'I am A2', myAwesomeData: 'AwesomeA2'},
            {id: 'A3', parent_id: 'A1', myAwesomeText: 'I am A3', myAwesomeData: 'AwesomeA3'},
            {id: 'B1', parent_id: null, myAwesomeText: 'I am B1', myAwesomeData: 'AwesomeB1'},
            {id: 'C1', parent_id: null, myAwesomeText: 'I am C1', myAwesomeData: 'AwesomeC1'},
            {id: 'C2', parent_id: 'C1', myAwesomeText: 'I am C2', myAwesomeData: 'AwesomeC2'}];
        var args = {id: 'id', parentId: 'parent_id', components: {checkbox: []}};

        $('#demo').tree(dataSource, template, args, function (data, i) {
                $(this).on($.flextree.components.checkbox.eventCheck, function () {
                    var checked = 'You checked:\n';
                    $(this).treeChecked().each(function () {
                        checked += $(this).text() + '\n';
                    });
                    alert(checked);
                });
            });

  • Ajax
    this component extended $.fn and added some new features, it don't need to be declared in components option.
    this component needs FontAwesome, please include it first.

    Template
    this component requires some specific elements in your template.

    • <span class="fa ft-ajax"></span>

    APIs

    • $().treeAjax(ajax)
      ajax {object}
      bind an ajax object to this node, once this node is activated, $.ajax(ajax) will be executed, so argument "ajax" is the same as $.ajax(ajax)'s ajax argument. This method returns a $.Deferred object.
            var template = '<a href="javascript:;">{myAwesomeText}</a><span class="fa ft-ajax"></span>';
        var dataSource = [{id: 'A1', parent_id: null, myAwesomeText: 'I am A1', myAwesomeData: 'AwesomeA1'},
            {id: 'A2', parent_id: 'A1', myAwesomeText: 'I am A2', myAwesomeData: 'AwesomeA2'},
            {id: 'A3', parent_id: 'A1', myAwesomeText: 'I am A3', myAwesomeData: 'AwesomeA3'},
            {id: 'B1', parent_id: null, myAwesomeText: 'I am B1', myAwesomeData: 'AwesomeB1'},
            {id: 'C1', parent_id: null, myAwesomeText: 'I am C1', myAwesomeData: 'AwesomeC1'},
            {id: 'C2', parent_id: 'C1', myAwesomeText: 'I am C2', myAwesomeData: 'AwesomeC2'}];
        var args = {id: 'id', parentId: 'parent_id', collapsible: true};

        $('#demo').tree(dataSource, template, args, function (data, i) {
            $(this).treeAjax({url: 'assets/i/data.json'}).done(function (node, data) {
                alert('Ajax done, now will add some new nodes from ajax data.');
                $(node).tree(data, template);
            }).fail(function (node, data) {
                alert('Ajax fail.');
            });
        });

  • Search
    this component extended $.fn and added some new features, it don't need to be declared in components option.

    APIs

    • $().treeSearch(keyword, selector)
      keyword {string}
      selector {string, optional}
      search this tree's text content and return matched nodes.
            var template = '<a href="javascript:;">{myAwesomeText}</a>';
        var dataSource = [{id: 'A1', parent_id: null, myAwesomeText: 'I am A1', myAwesomeData: 'AwesomeA1'},
            {id: 'A2', parent_id: 'A1', myAwesomeText: 'I am A2', myAwesomeData: 'AwesomeA2'},
            {id: 'A3', parent_id: 'A1', myAwesomeText: 'I am A3', myAwesomeData: 'AwesomeA3'},
            {id: 'B1', parent_id: null, myAwesomeText: 'I am B1', myAwesomeData: 'AwesomeB1'},
            {id: 'C1', parent_id: null, myAwesomeText: 'I am C1', myAwesomeData: 'AwesomeC1'},
            {id: 'C2', parent_id: 'C1', myAwesomeText: 'I am C2', myAwesomeData: 'AwesomeC2'}];
        var args = {id: 'id', parentId: 'parent_id'};

        var $tree = $.flextree.tree(dataSource, template, args);
        var $input = $('<input id="demo-search-kw"/>').val('A');
        var $button = $('<button>Search</button>').on('click', function () {
            var search = 'You got:\n';
            $tree.treeSearch($('#demo-search-kw').val()).each(function () {
                search += $(this).text() + '\n';
            });
            alert(search);
        });
        $('#demo').append($input).append($button).append($tree);
